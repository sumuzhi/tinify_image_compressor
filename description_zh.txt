Tinify 图片压缩工具

Tinify 图片压缩工具是一款基于 Tinify API (TinyPNG/TinyJPG) 的高效浏览器插件，帮助您轻松地在本地批量压缩图片，无需上传到第三方服务器，保护您的隐私安全。

主要功能：

- 智能无损压缩：利用 Tinify 先进的压缩算法，在肉眼几乎无法察觉差异的情况下，大幅减小图片体积（通常减少 50-80%）。
- 批量处理：支持一次性拖拽或选择多个文件/文件夹进行压缩。
- 隐私安全：图片仅通过 Tinify 官方 API 处理，不经过任何中间服务器，压缩完成后直接下载回本地。
- 本地化存储：您的 API Key 安全保存在本地浏览器中。
- 支持多种格式：完美支持 WebP、PNG 和 JPEG 图片格式。
- 一键下载：支持单个文件下载或一键打包下载所有压缩后的图片（ZIP 格式）。

使用指南：

1. 获取 API Key：
   访问 TinyPNG 开发者页面 (https://tinypng.com/developers) 注册并获取免费的 API Key（每月免费 500 张）。
2. 设置插件：
   打开插件，在输入框中粘贴您的 API Key。
3. 开始压缩：
   点击或拖拽图片文件到上传区域。
   点击“开始压缩”按钮。
4. 下载结果：
   压缩完成后，您可以查看每个文件的压缩率。
   点击“全部下载”即可获取压缩后的图片包。

隐私声明：

本插件承诺不收集任何用户个人信息。所有的图片处理均通过加密连接直接与 Tinify API 通信。您的 API Key 仅存储在您本地的浏览器缓存中。
